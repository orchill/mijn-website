Project 4: Home - Server 2025

Projectomschrijving

Dit project is een persoonlijk hobbyproject waarin ik in 2025 ben begonnen met het opzetten, configureren en uitbreiden van mijn eigen home server. Het doel was om een stabiel, veilig en volledig zelfgehost systeem te creëren waarop ik verschillende services kan draaien, zoals media-servers, monitoringtools, automatisaties en personal utilities. Tijdens het proces heb ik de server stap voor stap opgebouwd: ik heb services gecontaineriseerd, de netwerkinstellingen geoptimaliseerd en een duidelijke mappenstructuur en configuratiebeheer opgezet. Daarnaast heb ik gewerkt met verschillende tools om de server overzichtelijk, uitbreidbaar en betrouwbaar te houden. Dit home-serverproject gaf me de ruimte om te experimenteren met nieuwe technologieën, veilige configuraties en automatisaties. Het eindresultaat is een flexibele, stabiele en schaalbare omgeving die ik voortdurend blijf uitbreiden naarmate mijn behoeften groeien.

Kenmerken

  • 24/7 operationeel en stabiel systeem
  • Ondersteunt meerdere self-hosted services tegelijk.
  • Eenvoudig te beheren en automatiseren via dashboards en scripts.

Gebruikte technologieën

  • Linux
  • Docker
  • Jellyfin

Wat waren de verwachtingen?

Bij de start van dit project wilde ik vooral mijn basiskennis rond servers en self-hosting opfrissen. Ik wilde leren hoe je een moderne home-serveromgeving opbouwt met gecontaineriseerde services, reverse proxies en veilige toegangsinstellingen. Het project moest vooral een leeromgeving zijn waarin ik nieuwe services kon testen, infrastructuurtechnieken kon oefenen en mijn kennis van system administration verder kon uitbreiden.

Wat heb ik geleerd?

Tijdens dit project heb ik veel geleerd over het opzetten en beheren van een zelfgehoste serveromgeving. Ik ontdekte hoe belangrijk het is om services goed te isoleren, bijvoorbeeld via containers, en hoe je een stabiele en betrouwbare infrastructuur opbouwt die 24/7 kan draaien. Daarnaast heb ik inzicht gekregen in netwerkconfiguraties, beveiliging en toegangsbeheer, zoals het correct instellen van firewalls en veilige authenticatie. Ook heb ik ervaren hoe waardevol monitoring en logging zijn om problemen vroegtijdig op te sporen en de server soepel te laten functioneren. Tot slot heeft dit project mijn vaardigheden in automatisering, systeembeheer en het integreren van meerdere services verder ontwikkeld, waardoor ik nu efficiënter en gestructureerder kan werken aan self-hosted oplossingen.
