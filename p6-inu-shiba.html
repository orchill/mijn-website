Project 6: Inu Shiba - Website 2024

Projectomschrijving

Dit project was een tribute website voor Inu Shibas, ontwikkeld in 2024 voor een vriend. Het was een front-end website met een introductiepagina over Inu Shibas en een overzicht van hun karakteristieke eigenschappen, foto's en leuke feitjes. Tijdens het project heb ik me volledig gericht op ontwerp en gebruikerservaring, waarbij ik een overzichtelijke lay-out en visueel aantrekkelijke presentatie creëerde die de charme en speelsheid van Inu Shibas weerspiegelde. Doordat ik toen al wat meer kennis had van HTML, CSS en JavaScript, kon ik de website net iets uitgebreider en verzorgder maken dan mijn eerdere projecten. Het resultaat is een eenvoudige maar effectieve website die bezoekers direct laat zien waarom Inu Shibas zo geliefd zijn.

Kenmerken

  • Cross-browser compatibiliteit
  • Eén consistent thema

Gebruikte technologieën

  • HTML5
  • CSS3
  • JavaScript

Wat waren de verwachtingen?

De verwachting voor dit project was om een overzichtelijke en nette website te maken die de Inu Shibas op een aantrekkelijke manier presenteert. Er werd gevraagd om een simpel en consistent kleurenschema, zonder complexe of afleidende kleurkeuzes, zodat de inhoud en afbeeldingen van de honden centraal stonden.

Wat heb ik geleerd?

Tijdens dit project heb ik geleerd hoe ik een duidelijke en functionele header kan ontwerpen die de navigatie overzichtelijk maakt. Daarnaast heb ik mijn vaardigheden in het schrijven van nette en gestructureerde code verder verbeterd, wat de onderhoudbaarheid en leesbaarheid van de website ten goede kwam.
